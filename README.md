# 30 Days Gcloud Dashboard

### Under development

### Installations
* Requirements 

    `pip3 install -r requirements.txt`

* Create the database by running main.py in scrapper
   
      cd scrapper
      python3 main.py
   
 * Create flask app 

	  `export FLASK_APP=server.py`
  
* Run flask app
	
	 `flask run`
